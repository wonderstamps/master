WonderStamps

Project Summary

The WonderStamps project is series of 8 generative, non fungible token (NFT) art collections featuring Wonders of the World. The first series in the collection (The Pyramids of Giza in Egypt) was add as an 8th "honorary wonder" as it is the only surviving original wonder.

Each art piece in the project is unique and tokenized on the Ethereum blockchain as a single edition token. The elements in each series are programatically assembled using varying degrees of probability to create each image. This process gives each piece a unique look which we ultimately had no control over. There might even be some super rare elements sprinkled in periodically to keep an eye out for. We hope you enjoy browsing the entire series to discover the many elements used throughout each series.

We have been inspired by the beauty and awe of each of these wonders. Our goal with our art is to inspire others to visit these amazing structures. Not only do we want to inspire with this project, but we also want to give back.

We are committed to giving a portion of the initial proceeds for each series sales directly to charites helping around the world. Prior to each series release a charity will be assigned as well as the percentage of initial sale proceeds to be donated. The donation tracker on this website keep track of the amount donated for each series.

The original art image files will be uploaded to the InterPlanetary File System (IPFS) for uniqueness verification and preservation.

Please enjoy the art, join the community, be inspired, and help those in need.

Art Elements

Each unique art piece is generated dynamicaly from multiple layers.

  • Stamp Border Color
  • Sky
  • Wonder of the World
  • Ground
  • Foreground
  • Local Elements
  • Other Rare Elements...

Creation Process

The base image elements (stamp border, sky, wonder, ground, foreground, local & rare elements) were created first.

The overlay colors / patterns were designed and laid over the top of the base image elements creating multiple versions of each element.

Build Process

Using a custom built python script, each individual art piece is generated by layering the elements on top of the previous.

Each element was given a unique probability of occurence which allows for element rarity to be built into the art programatically.

This approach for determining the combination of elements randomizes the look as well as the uniqueness of the elements used in each piece of art.

Local Elements

Each series features endangered or threatened plants and animals local to that location. Our goal is to bring awareness to conservation efforts for these important species.

Keep an eye out for other elements that highlight that series. Some may be harder to spot than others.

Charity

We are committed to donating a portion of the initial sales from each series to a charity helping around the world. The donation level and the charity will be announced for each series prior to series release.

Track each series charity donations using our donation tracker.
