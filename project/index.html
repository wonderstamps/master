Art Elements

Each unique art piece is generated dynamicaly from the following layers.

  • Base stamp
  • Border Color
  • Sky + Optional Overlay
  • Pyramids + Optional Overlay
  • Sand + Optional Overlay
  • Local Element
  • Other Rare Elements...

Creation Process

The base image elements (stamp, border, sky, wonder, ground, local & rare) were created first.

The overlay colors / patterns were designed and laid over the top of the base image elements creating multiple versions of each element.

Build Process

Using a computer script, each art piece is generated by layering each element on top of the previous starting with the base stamp layer.

Each layer was given a probability of being chosen for each image.

This approach for determining the combination of elements randomizes the look as well as the uniqueness of the elements used in each piece of art.

Local Elements

Each series features local endangered plants and animals.

Our goal is to bring awareness to conservation efforts.

Local Charity

A portion of the sales from each series will go to a local charity in that country.
